Key Findings

Entity Fragmentation

Legacy OCR creates 21% more nodes due to OCR errors fragmenting entities. "America" becomes "Ainerica", "Aineriea", etc.

Graph Fragmentation

Legacy has 80% more disconnected components (305 vs 169). Noisy text breaks the knowledge graph into isolated islands.

Orphan Nodes

Legacy has 67% more orphan nodes (14.2% vs 8.5%). These isolated entities are essentially useless for analysis.

LLM Robustness

Gemini semantic extraction is surprisingly robust - similar relation counts from noisy text. But entity names still inherit OCR errors.
