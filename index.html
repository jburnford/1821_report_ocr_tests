Key Findings: OCR Quality Impact

Entity Fragmentation

Legacy OCR creates 21% more nodes due to OCR errors splitting entities. "America" becomes "Ainerica", "Aineriea", etc.

Graph Fragmentation

Legacy has 80% more disconnected components (305 vs 169). OCR noise breaks the knowledge graph into isolated islands.

Orphan Nodes

Legacy has 102% more orphans (253 vs 125). These isolated entities are useless for graph analysis.

Garbage Entities

OCR errors create nonsense nodes like "tlie 'Comniittee", "December x795", "Miramiclii iii" that pollute the knowledge graph.
